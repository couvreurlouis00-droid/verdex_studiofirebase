# **App Name**: Verdex Nexus

## Core Features:

- Dynamic UI Rendering: Reconstruct the existing site's intricate layout and interactive elements using Next.js and React components for enhanced modularity and performance, moving beyond static HTML.
- Real-time Protocol Metrics: Display live blockchain metrics (TPS, CO2 offset, renewable coverage, green efficiency) by integrating with a dedicated backend API to provide actual network data, replacing client-side simulations.
- Real-time VDX Price Feed: Integrate with a backend service to fetch and display the live VDX token price, including a dynamic mini-chart, ensuring up-to-date market information.
- Secure Waitlist Management: Implement a robust backend API to securely receive and store waitlist form submissions (full name, email, role) into a database, providing actual user management.
- Airdrop Eligibility Checker: Develop a backend service that validates submitted wallet addresses for simulated airdrop eligibility, replacing the client-side deterministic logic.
- AI-Powered Eligibility Feedback Tool: Leverage a generative AI tool to provide personalized and detailed feedback on wallet eligibility results, offering specific guidance and insights on how to improve a user's simulated blockchain-environmental footprint or enhance future allocation chances.
- Centralized Content Management: Serve dynamic content sections like the FAQ and Roadmap from a backend API, enabling administrators to easily update these sections without requiring frontend code changes.

## Style Guidelines:

- Primary color: A vibrant, clear green (#3dff7a) to symbolize growth and sustainability, providing strong contrast on dark backgrounds.
- Background color: A very deep, muted forest green (#121914), offering a sophisticated and eco-conscious base for the design.
- Accent color: A soft, pastel yellow-green (#9de58c) used for subtle highlights and secondary elements, maintaining an organic feel while providing visual diversity.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, techy, and bold display.
- Body font: 'Literata' (serif) for a sophisticated, literary, and easy-to-read textual experience.
- Code and monospace text font: 'Source Code Pro' (monospace) for displaying data points and technical information with clarity.
- Maintain the use of subtle, geometric icons like hexagrams (⬡) and circles, consistent with the existing visual language for blockchain elements.
- Emphasize clean, structured grid layouts with ample spacing, maintaining the minimalist and professional feel for optimal content hierarchy and readability.
- Refine existing scroll-reveal animations for sections like features and roadmap, ensuring smooth transitions and a dynamic, engaging user experience.
